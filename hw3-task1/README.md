# HW3 Task 1

这个文件夹保留任务 1 的必要代码、配置和复现说明，不包含大体积结果文件。

## Included Folders

- [`fusion/`](fusion/README.md)：场景融合脚本与说明。
- [`objectA/`](objectA/README.md)：物体 A 的 2DGS 训练与后处理代码。
- [`objectB/`](objectB/README.md)：物体 B 的 threestudio 配置与导出相关代码。
- [`objectC/`](objectC/README.md)：物体 C 的 Magic123 配置、脚本与兼容性代码。

## Not Included

- 训练权重
- 渲染结果
- 点云与网格输出
- 中间缓存
- 评估图表与日志导出
